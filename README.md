# 🤖 Python RPG Bot

Python Dev Quest is a small educational chatbot project built with Python.

The goal of this project is to help beginners learn programming through a simple interactive system with quizzes, motivation, and chatbot conversations.

## 🚀 Features

- 🤖 Programming assistant chatbot
- 🎮 Programming quiz challenges
- 🏆 Score system
- 💬 Motivational chatbot
- 🎲 Random programming questions

## 📚 What I Learned

This project helped me practice:

- Python modular programming
- Dictionaries and lists
- Random module
- Basic chatbot logic
- Organizing a project with multiple files

